1. 加入测试群组：https://groups.google.com/g/ruiqianyi
2. 安装应用：https://play.google.com/store/apps/details?id=io.mybatis.classcheckin

## 功能特点

### 📚 课程管理
- 创建和管理多个课程
- 记录课程名称、学员列表、总课时数
- 实时显示已用课时和剩余课时
- 支持课程编辑和删除

### 📅 日历打卡
- 直观的月视图日历界面
- 点击日期快速记录打卡
- 支持多学员同时打卡
- 已打卡日期自动标记（绿色显示）
- 可为每次打卡添加备注

### 📊 统计分析
- 自动统计已用课时和剩余课时
- 按日期倒序显示打卡记录
- 查看每次打卡的详细信息
- 支持编辑和删除历史打卡记录
